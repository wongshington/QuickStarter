# QuickStarter

[Check Me Out Live!](https://quick-starter.herokuapp.com/)

QuickStarter is a project crowd funding web application, inspired by Kickstarter. It is a full-stack app built using a `Ruby on Rails` backend, `PostgreSQL` database, and `React/Redux` front-end architecture.

## Features

- Project creation
- Support your favorite projects
- Search for the next great project
-
